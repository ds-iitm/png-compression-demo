Lossless PNG Compression Demo

This project demonstrates lossless compression of PNG images to sizes under strict byte limits using Python and external tools like optipng and Pillow.

Features

  • Lossless compression (every pixel preserved)
  • Optimized output for minimal size
  • Script-based and reproducible
